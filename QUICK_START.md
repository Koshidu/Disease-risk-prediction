# 🚀 Quick Start Guide

## 3 Simple Steps to Get Started

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Train the Model
```bash
python train_model_max.py
```
⏱️ This takes 1-2 minutes. Wait for "✅ Model training completed successfully!"

### Step 3: Run the Application
```bash
python app.py
```
Then open: **http://localhost:5000**

---

## 🎯 What You Can Do

1. **Fill in the form** with patient information
2. **Click "Predict Disease Risk"**
3. **View results:**
   - Predicted disease with confidence score
   - Recommended specialist
   - Natural remedies with detailed information
